# Portfolio

## Description

This will be my personal portfolio. SPA client, monolithic backend utlizing Pocketbase backend platform :)

## Development Environment

**Pre-requisites**

- Vagrant
- VirtualBox
- HOST OS: Windows

## Guide

1.		git clone $repo_url && Vagrant up
2. Point a web browser on your host machine to ```http://10.0.0.1:7001``` to check if Nginx was installed correctly 

- Note: By default the development environment is configured with a static IPv4 address set to ```10.0.0.1``` which exposes port 7001 to your host OS on your *internal network*. You may change these values within the Vagrantfile.
