# Portfolio

## Description

This will be my personal portfolio. It will consist of an SPA client with a monolithic backend that will I plan on running on my own infrastructure.

## Getting Started - Development Environment

## Pre-requisites

- Vagrant
- VirtualBox

## Guide

1.		git clone $repo_url && Vagrant up
2. Point a web browser on your host machine to ```http://10.0.0.1:7001``` to check if Nginx was installed correctly 

- Note: By default the development environment is configured with a static IPv4 address set to ```10.0.0.1``` which exposes port 7001 to your host OS. You may change these values within the Vagrantfile
